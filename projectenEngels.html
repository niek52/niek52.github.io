Projects

Mastermind:

Content:
Mastermind is a classic among the games in logical thinking. One player tries to unravel the secret code of the other player. The codemaker secretly chooses four code pawns and places them in a chosen order behind the screen. It is permissible to choose two or more pawns of the same color. The codemaker has the choice of six different colors. This is the hidden code that the codebreaker must unravel.

During my education, we recreated this game as a project.


Drupal Project Aoste

Content:
In the third year of my education, we sat down with Aoste for our project to set up a website in Drupal. This website had to allow a user to consult products and make use of these products to consult in which stores they are available. In addition, the website had to allow a user to enable a user to use discount codes, create an account and log in and get detailed information about the products. A user could then choose a product from a particular retailer and he would be redirected to that product on the website of the chosen retailer where he could then make further purchases.
