1. Name of the Company:

SecuryKey INC.


2. Name of the Product:

SecuryKey_Mate SecuryKey_summa

3. Product description:

The SecuryKey_mate version is the first product of our company. This USB with RAM has a chance to revolutionize the use of portable storage devices. It will be using Ram memory, a battery and a mini-PC. Because of the combination of RAM memory and USB 3.2 we will be able to read and write data about 20 times faster than a normal USB-drive. We also use a lesser used feature of RAM memory: if you turn of the power of your device all the data is automatically wiped. This is very useful for the military/financial practices were you can’t have the risk of data falling in the wrong hands. The SecuryKey_summa version is the second version. This version has an integrated wifi network that enables the user to share data with everyone that is connected to the network. This item could be of use in multiple occaisions. For example, when several people need to see the same information in a meeting. instead of using something like github, google, docs they just have to connect and download the files that are on the USB.

4. Components:

-RAM
-Battery 5v
-Lcd-Screen
-USB 3.2 port (to quickly transfer data)
-Micro USB port to charge the battery


5. Additional value to society

The securykey_mate is developed in a way that’s it affordable for practically, everyone. Everybody who needs to transport huge amounts of data should be able to use and afford this just fine, its development provides security to either businessman, that carry around sensitive company data, or just the regular household who don’t want their private data to be exposed to the public. The somewhat older generation that missed the whole “cloud” thing can also find practical use for this, this way of transporting data will be “revived” for them since they only had this transportation method back then.


6. Target group

The groups that we want to sell our product to are mainly military and financial because of the kind of data they work with. But also tech savvy people can benefit from this product because of the speeds that can be acquired. And everybody got some things they want to store that they don't want the rest of the world to see.

7. Market share

We have done our research and we didn’t find any product that does the same as ours.
